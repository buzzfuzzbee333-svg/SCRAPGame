# SCRAPGame

## Project docs
- [Last Scrap Master Spec](docs/LAST_SCRAP_MASTER_SPEC.md)
- [Last Scrap Architecture](docs/LAST_SCRAP_ARCHITECTURE.md)

## Current direction
Last Scrap is a compact zombie survival + scrap-risk game built around:
- defending a central rig
- surviving short enemy waves
- earning unsecured scrap during a run
- choosing to cash out or continue
- permanent upgrades between runs

## Locked version-1 rules
- Contact/proximity zombie attacks only
- No ranged/spit attacks yet
- Rig overrun causes partial unsecured scrap loss
- Player death causes total unsecured scrap loss
- The game should be built as a clean vertical slice before expanding scope
